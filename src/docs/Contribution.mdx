# Storybox

Welcome to our design system documentation! This comprehensive guide is designed to provide you with all the information you need to create cohesive and consistent user experiences across all of our products and platforms.

Our design system is a collection of guidelines, principles, and assets that help developers create effective, accessible, and user-friendly interfaces. By following these guidelines, we can ensure that our products are easy to use, visually appealing, and consistent, regardless of who is working on them or which platform they are being used on.

This documentation will walk you through our design system's components.

## Contributing
Before contributing, get to know our [Code of Conduct](/CODE_OF_CONDUCT.md)

You can then contribute to this project by following those simple steps:

1. Fork this repository
2. Clone the forked repository
3. Create a new branch
4. Make your changes
5. Commit and push your changes
6. Create a pull request
