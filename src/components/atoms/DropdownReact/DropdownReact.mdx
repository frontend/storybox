Dropdown (react) is a React widget. To use it in your project, follow these steps:

**1**. Include the styles in the `<head>`:

<Source language="html" dark code={`
<link rel="stylesheet" href="styles.css">
`}/>

**2**. Include React and the following bundle before `<body>` tag's closing:

<Source language="html" dark code={`
<script
  src="https://cdnjs.cloudflare.com/ajax/libs/react/18.2.0/umd/react.production.min.js"

  referrerpolicy="no-referrer"

`}/>

<Source language="html" dark code={`
